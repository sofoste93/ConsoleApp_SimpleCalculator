# ConsoleApp_SimpleCalculator
A simple calculator with C#: Version 1.0

--------------------------------------------------------

# In this version

- User input: On
- Wrong user input: off
- Select arithmetic symbole: off
- Basic operation: 
  Add, 
  subtract, 
  multiplicate,
  divide
- Restart operation: off
- Auto close: off
